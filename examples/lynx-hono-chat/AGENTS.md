## Commands

- `npm run build` - TypeScript solution build for `script`, `src/server`, and `src/client`

- `npm run typecheck` - TypeScript solution typecheck for the workspace

- `npm run dev` - Build the workspace, then start the CLI which launches the server and builds the client bundle

- `npm run build:client` - Build the Lynx client bundle with Rspeedy
